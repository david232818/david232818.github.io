The "tiny DES", or TDES, program.

The file outDiff contains pairs of chosen plaintexts and the 
corresponding ciphertexts. This can be used to implement the 
differential attack discussed in the text.

The file outLin contains chosen plaintexts and the corresponding 
ciphertexts. This can be used to implement the linear attack 
discussed in the text.

In Windows, the files outDiff and outLin can be viewed using TextPad.

This crypto algorithm is very insecure!
