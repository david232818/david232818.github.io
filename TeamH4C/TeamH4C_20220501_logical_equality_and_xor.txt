---[ 1. 1050번 문제
 [1]은 다음과 같이 문제를 제시한다.

-------------------------------------------------------
두 정수(a, b)를 입력받아
a와 b가 같으면 1을, 같지 않으면 0을 출력하는 프로그램을 작성해보자.

입력
두 정수 a, b가 공백을 두고 입력된다.
-2147483648 <= a, b <= +2147483647


출력
a와 b의 값이 같은 경우 1을, 그렇지 않은 경우 0을 출력한다.
-------------------------------------------------------

---[ 2. 문제 풀이
 이 문제는 논리적인 관계 연산자인 "==" 또는 "!="를 사용하여 풀 수도 있지만
Bitwise XOR의 특성을 활용하여 풀 수도 있다. 이는 다음과 같이 구현될 수 있다.

-------------------------------------------------------
#include <stdio.h>
#include <stdint.h>

int main()
{
    int32_t a, b, res;
    
    scanf("%d %d", &a, &b);
    if (a ^ b)
        res = 0;
    else
        res = 1;
    printf("%d", res);
    return 0;
}
-------------------------------------------------------
