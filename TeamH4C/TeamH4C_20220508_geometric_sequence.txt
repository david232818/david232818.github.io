---[ 1. 1090번 문제
 [1]은 다음과 같이 문제를 제시한다.

-------------------------------------------------------
수학에서는 앞뒤 수들의 비율이 같다고 하여
등비(비율이 같다의 한문 말) 수열이라고 한다.

시작 값(a), 등비(r), 몇 번째인지를 나타내는 정수(n)가 입력될 때
n번째 수를 출력하는 프로그램을 만들어보자.

입력
시작 값(a), 등비의 값(r), 몇 번째 인지를 나타내는 정수(n)가
공백을 두고 입력된다.(모두 0 ~ 10)


출력
n번째 수를 출력한다.
-------------------------------------------------------

---[ 2. 문제 풀이
 [1]이 제시하는 등비 수열의 정의로부터 등비 수열은 다음과 같이 진행됨을
알 수 있다.

-------------------------------------------------------
a, ar, ar^2, ar^3, ... ar^(n - 1)
1  2   3     4     ... n
-------------------------------------------------------

따라서 등비 수열의 일반항을 ar^(n - 1)이라고 생각할 수 있고, [1]이 제시하는
입력값의 제한을 생각하면 다음과 같은 코드를 작성할 수 있다.

-------------------------------------------------------
#include <stdio.h>

int main()
{
    int a, r, n, i;
    long long r_n_1;
    
    scanf("%d %d %d", &a, &r, &n);
    r_n_1 = 1;
    for (i = 1; i != n; i++)
        r_n_1 = r_n_1 * r;
    printf("%lld\n", (long long) a * r_n_1);
    return 0;
}
-------------------------------------------------------
