---[ 1. 1097번 문제
 [1]은 다음과 같이 문제를 제시한다.

-------------------------------------------------------
부모님을 기다리던 영일이는 검정/흰 색 바둑알을 바둑판에 꽉 채워 깔아 놓고 놀다가...

"십(+)자 뒤집기를 해볼까?"하고 생각했다.

바둑판(19 * 19)에 흰 돌(1) 또는 검정 돌(0)이 모두 꽉 채워져 놓여있을 때,
n개의 좌표를 입력받아 십(+)자 뒤집기한 결과를 출력하는 프로그램을 작성해보자.

입력
바둑알이 깔려 있는 상황이 19 * 19 크기의 정수값으로 입력된다.
십자 뒤집기 횟수(n)가 입력된다.
십자 뒤집기 좌표가 횟수(n) 만큼 입력된다. 단, n은 10이하의 자연수이다.


출력
십자 뒤집기 결과를 출력한다.
-------------------------------------------------------

---[ 2. 문제 풀이
 [1]이 제시한 문제에서 설명하는 십자 뒤집기는 주어진 x, y 좌표에 대해 x를
가로축 값으로 가지는 모든 세로축의 값이 0이면 1로, 1이면 0으로 뒤집고, y를
세로축 값으로 가지는 모든 가로축의 값이 0이면 1로, 1이면 0으로 뒤집는 것이다.
이를 2차원 배열로 설명하면 주어진 x, y에 대해 x열의 모든 값을 0이면 1로,
1이면 0으로 뒤집고, y행의 모든 값을 0이면 1로, 1이면 0으로 뒤집는 것이다.
이러한 십자 뒤집기는 다음과 같은 코드로 구현할 수 있다.

-------------------------------------------------------
#include <stdio.h>

int main()
{
    int n, i, j, board[19][19];
    
    for (i = 0; i != 19; i++)
        for (j = 0; j != 19; j++)
            scanf("%d", &board[i][j]);
    scanf("%d", &n);
    while (n != 0) {
        int x, y;
        
        scanf("%d %d", &x, &y);
        x--;
        y--;
        for (i = 0; i != 19; i++) {
            if (board[i][y])
                board[i][y] = 0;
            else
                board[i][y] = 1;
        }
        for (j = 0; j != 19; j++) {
            if (board[x][j])
                board[x][j] = 0;
            else
                board[x][j] = 1;
        }
        n--;
    }
    for (i = 0; i != 19; i++) {
        for (j = 0; j != 19; j++)
            printf("%d ", board[i][j]);
        printf("\n");
    }
    return 0;
}
