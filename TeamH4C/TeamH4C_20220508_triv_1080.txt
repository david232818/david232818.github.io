---[ 1. 1080번 문제
 [1]은 다음과 같이 문제를 제시한다.

-------------------------------------------------------
1, 2, 3 ... 을 계속 더해 나갈 때,
그 합이 입력한 정수(0 ~ 1000)보다 같거나 작을 때까지
계속 더하는 프로그램을 작성해보자.

즉, 1부터 n까지 정수를 계속 더한다고 할 때,
어디까지 더해야 입력한 수보다 같거나 커지는지 알아보고자 하는 문제이다.

입력
정수 1개가 입력된다.


출력
1, 2, 3, 4, 5 ... 를 순서대로 계속 더해 합을 만들어가다가,
입력된 정수와 같거나 커졌을 때, 마지막에 더한 정수를 출력한다.
-------------------------------------------------------

---[ 2. 문제 풀이
 [1]이 제시한 문제는 주어진 정수보다 1부터 n까지 더한 합이 작은 '동안'에는,
즉 주어진 정수가 1부터 n까지 더한 합보다 큰 '동안'에는 계속 덧셈을 진행하라는
문제로 볼 수 있다. 그러므로 while statement를 사용하여 풀 수 있다.

-------------------------------------------------------
#include <stdio.h>

int main()
{
    int n, i, sum;
    
    scanf("%d", &n);
    i = sum = 0;
    while (n > sum) {
        i++;
        sum += i;
    }
    printf("%d", i);
    return ;
}
-------------------------------------------------------
