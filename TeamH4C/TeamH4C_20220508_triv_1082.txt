---[ 1. 1082번 문제
 [1]은 다음과 같이 문제를 제시한다.

-------------------------------------------------------
A, B, C, D, E, F 중 하나가 입력될 때,
1부터 F까지 곱한 16진수 구구단의 내용을 출력해보자.
(단, A ~ F 까지만 입력된다.)

입력
16진수로 한 자리 수가 입력된다.
단, A ~ F 까지만 입력된다.


출력
입력된 16진수에 1~F까지 순서대로 곱한, 16진수 구구단을 줄을 바꿔 출력한다.
계산 결과도 16진수로 출력해야 한다.
-------------------------------------------------------

---[ 2. 문제 풀이
 [1]이 제시한 문제의 입력값의 제한 조건에 의해 입력되는 것은 숫자 또는 A-F의
문자이고 한 자리이다. 따라서 숫자 문자가 아닌 문자가 입력된다면 그것은 A-F일
것이고, 이외의 경우에는 모두 숫자 문자가 입력될 것이다. 여기서 숫자 문자의
경우에는 C의 기본 문자 집합인 ASCII에 의해 '0'을 빼서 숫자로 변환할 수 있다.
하지만 A-F의 경우에는 'A'를 빼는 것만으로는 부족하고 16진법의 규칙에 따라
10을 더해주어야 한다.

 [2, Sec. 7.4.1.2]는 isalpha 함수의 기능을 주어진 문자가 isupper또는
islower를 참으로 만드는지 검사하는 것이라고 설명한다. 그리고
[2, Sec. 7.4.1.11]과 [2, Sec. 7.4.1.7]은 각각 isupper 함수와
islower 함수를 설명하며 isupper 함수는 오직 대문자 알파벳 또는
locale-specific한 문자로 iscntrl, isdigit, ispunct, isspace 함수를
참이 아니도록 만드는 (대문자) 문자에 대해 참을 반환하고, islower 함수는 오직
소문자 알파벳 또는 locale-specific한 문자로 iscntrl, isdigit, ispunct,
isspace 함수를 참이 아니도록 만드는 (소문자) 문자에 대해 참을 반환한다고
설명한다. 이때 [2, Sec. 7.4.1.4], [2, Sec. 7.4.1.5],
[2, Sec. 7.4.1.9], [2, Sec. 7.4.1.10]은 각각 iscntrl, isdigit,
ispunct, isspace 함수를 설명하며 iscntrl 함수는 제어 문자에 대해 참을
반환하고, isdigit 함수는 10진수 문자에 대해 참을 반환하고, ispunct 함수는
출력을 위한 문자 중 isspace 함수와 isalnum 함수를 참이 아니도록 만드는
문자에 대해 참을 반환하고, isspace 함수는 표준 공백 문자 또는 isalnum 함수를
참이 아니도록 만드는 문자에 대해 참을 반환한다. 그리고 isalnum 함수는
[2, Sec. 7.4.1.1]가 설명하는데 isalpha와 isdigit을 참으로 만드는 문자에
대해 참을 반환한다. 따라서 isalpha 함수를 통해 숫자 문자가 아닌 문자 여부를
판별할 수 있고 [1]이 제시한 문제에서는 A-F 여부를 판별할 수 있을 것이다.

-------------------------------------------------------
#include <stdio.h>
#include <ctype.h>

int main()
{
    char c;
    int i, n;
    
    scanf("%c", &c);
    if (isalpha(c))
        n = c - 'A' + 10;
    else
        n = c - '0';
    for (i = 1; i != 0x10; i++)
        printf("%X*%X=%X\n", n, i, n * i);
    return 0;
}
