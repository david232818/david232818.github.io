---[ 1. 1066번 문제
 [1]은 다음과 같이 문제를 제시한다.

-------------------------------------------------------
세 정수 a, b, c가 입력되었을 때, 짝(even)/홀(odd)을 출력해보자.

입력
세 정수 a, b, c 가 공백을 두고 입력된다.
0 <= a, b, c <= +2147483647


출력
입력된 순서대로 짝(even)/홀(odd)을 줄을 바꿔 출력한다.
-------------------------------------------------------

---[ 2. 문제 풀이
 이 문제에는 "어떻게 짝/홀을 판별할 것인가"와 "판별한 결과에 따른
출력방법"이라는 두 가지 문제가 있다. 먼저 후자에 대해서는 if-else문을
사용할 수도 있지만 판별 결과에 따라 "even"과 "odd"를 매핑한 테이블을
사용할 수도 있다. 그리고 전자에 대해서는 % 연산자를 사용할 수도 있지만
수를 이진법으로 표현했을 때 보여지는 특성에 대해 생각해보면 모든 홀수는
LSB가 1이고, 모든 짝수는 LSB가 0임을 알 수 있다. 이를 활용하여 짝/홀을
판별할 수 있을 것이다. 이로부터 다음과 같은 코드를 작성할 수 있다.

-------------------------------------------------------
#include <stdio.h>
#include <stdint.h>

int main()
{
    char *e_or_o[2] = { "even", "odd" };
    int32_t a, b, c;
    
    scanf("%d %d %d", &a, &b, &c);
    printf("%s\n%s\n%s\n", e_or_o[a & 1], e_or_o[b & 1], e_or_o[c & 1]);
    return 0;
}
