---[ 1. 1020번 문제
 [1]은 다음과 같이 문제를 제시한다.

-------------------------------------------------------
주민번호 앞 6자리와 뒷 7자리가 '-'로 구분되어 입력된다.
(입력값은 가상의 주민번호이다.)
ex)110011-0000000

'-'를 제외한 주민번호 13자리를 모두 붙여 출력한다.
-------------------------------------------------------

---[ 2. 문제 풀이
 입력 형식에서 "-" delimiter로하여 parsing하는 것은 scanf 함수로
처리할 수 있다. 그리고 이를 출력 형식에 맞추는 것은 %06d, %06d를
conversion specification으로 지정하면 해결될 것이다. 이로부터
다음과 같은 코드를 작성할 수 있다.

-------------------------------------------------------
#include <stdio.h>

int main()
{
	int l, r;

	scanf("%d.%d", &l, &r);
	printf("%06d%06d\n", l, r);
	return 0;
}
-------------------------------------------------------
